# v1.1.0

v1.0.0 リリース後の体験改善・品質強化をまとめたリリースです。

デモ: https://shunsukesaito00.github.io/mvmake/

## 新機能

- **サンプルプロジェクト**: オンボーディングから1クリックで、Canvas 生成のプレースホルダー画像4枚 + テキスト3種で構成されたサンプルを展開。メディアを持っていなくても編集・プレビュー・書き出しを体験できます
- **ショートカット一覧モーダル**: `?` キーまたはツールバーのヘルプボタンで表示

## 改善

- **PWA maskable アイコン**: マスク切り抜きで欠けない全面塗り + セーフゾーン配置の専用アイコンを追加
- **README スクリーンショット**: サンプルプロジェクトを使った実際の編集風景に更新(`npm run screenshot` で再生成可能)
- **CI/CD**: GitHub Pages デプロイ後に公開環境へのスモーク E2E を自動実行する `smoke` ジョブを追加

## バグ修正

- タイムラインのクリップ上部(ラベルと重なる領域)でトリムハンドルが掴めず「移動」になってしまう問題を修正

## テスト

- ユニットテスト 28件 → 36件(トリムのクランプ処理を追加)
- E2E 9シナリオ → 13シナリオ(サンプルプロジェクト、`?` キー、左右トリムを追加)
