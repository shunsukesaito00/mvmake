# v1.1.2

GitHub Pages デプロイの復旧と初回編集体験の改善リリースです。

デモ: https://shunsukesaito00.github.io/mvmake/

## 修正

- **Pages デプロイのタイムアウト連鎖を解消**: Deployments API だけでは検出できなかった `deployment_in_progress` 滞留を、Pages API でキャンセル・最大 5 分待機するよう修正。`#10` 以降 deploy が 10 分タイムアウトで失敗し本番が更新されない問題を解決
- **artifact 権限の明示**: Pages アップロード前に `chmod -cR +Xr dist` を追加し、権限起因のデプロイ失敗を防止

## 追加・改善

- **サンプルプロジェクト後の再生ガイド**: オンボーディングからサンプルを開いたあと、再生ボタンを 5 秒間ハイライトし「▶ 再生してプレビュー」を表示
- **空プロジェクト時の書き出し無効化**: クリップ 0 件では書き出しボタンを disabled にし、ツールチップで理由を表示
- **インスペクター未選択時のクイックスタート**: テキスト追加・メディアインポートへのショートカットを EmptyState 下に配置

## 運用上の注意

- main への **連続 push は避け**、1 回の push ごとに Deploy ワークフローの完了（build → deploy → smoke）を待ってから次を push してください
