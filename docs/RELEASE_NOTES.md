# v1.1.3

編集機能の拡張リリースです。音量キーフレームと書き出しプリセットを追加しました。

デモ: https://shunsukesaito00.github.io/mvmake/

## 追加

- **音量キーフレーム (MVP)**: AudioClip / VideoClip のクリップ内音量を時間軸上のキーフレームで変化させられます。インスペクターから追加・編集・削除可能
- **書き出しプリセット (MVP)**: 品質・解像度・In/Out 範囲を名前付きで保存し、書き出しダイアログからワンクリックで呼び出せます（localStorage 保存）
- **インスペクター未選択時のクイックスタート**: テキスト追加・メディアインポートへのショートカット

## 改善

- **本番スモーク E2E 拡充**: 音量キーフレームの動作を deploy smoke / `test:e2e:prod` で検証
- **書き出し UI**: 品質・解像度トグルに `aria-pressed` を追加し、適用状態をアクセシブルに

## 運用上の注意

- main への **連続 push は避け**、1 回の push ごとに Deploy ワークフローの完了（build → deploy → smoke）を待ってから次を push してください
