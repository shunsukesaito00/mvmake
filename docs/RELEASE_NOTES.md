# v1.2.0

SNS 向け縦型動画・構造化テンプレ・テキスト排版の拡張リリースです。結婚式ムービーの制作フローをさらに効率化しました。

デモ: https://shunsukesaito00.github.io/mvmake/

## 追加

- **縦型 9:16 プロジェクトプリセット (MVP)**: 1080×1920 を解像度プリセットに追加、書き出しラベル「9:16 で書き出し」対応
- **構造化ウェディングテンプレート拡張 (MVP)**: 「結婚式フル構成」テンプレートに章マーカー 5 件・写真ガイド 8 件を自動配置
- **テキスト行間・縦配置の詳細設定 (MVP)**: `lineHeight`（0.8〜2.5 倍）と `verticalAlign`（上/中央/下）をインスペクター・compositor に反映

## 改善

- **旧プロジェクト互換**: テキストクリップの `lineHeight` / `verticalAlign` は読み込み時にデフォルト値を補完
- **E2E**: 全 25 シナリオ（縦型書き出し・構造化テンプレ・行間/縦配置を追加）

## 運用上の注意

- main への **連続 push は避け**、1 回の push ごとに Deploy ワークフローの完了（build → deploy → smoke）を待ってから次を push してください

# v1.1.5

編集・書き出しの拡張リリースです。ルックプリセットから複数行テキストまで、結婚式ムービー制作の表現力を強化しました。

デモ: https://shunsukesaito00.github.io/mvmake/

## 追加

- **カラールックプリセット (MVP)**: ナチュラル / フィルム風 / セピア / ソフトなど 5 種をワンクリック適用
- **映像クリップのフェードイン/アウト (MVP)**: VideoClip / ImageClip の opacity を時間変化
- **プロジェクト解像度と書き出しの整合 (MVP)**: 4K / 正方形はネイティブ解像度で書き出し、720p はダウンスケール
- **テキスト複数行描画 (MVP)**: `\n` 改行をプレビュー・書き出し両方で描画

## 改善

- **書き出しプリセット**: 解像度を「プロジェクト / 720p」に整理（旧 1080p は互換正規化）
- **4K 書き出し**: 解像度に応じたビットレート自動調整
- **E2E**: 全 22 シナリオ（カラールック・映像フェード・正方形書き出し・複数行テキストを追加）

## 運用上の注意

- main への **連続 push は避け**、1 回の push ごとに Deploy ワークフローの完了（build → deploy → smoke）を待ってから次を push してください
