# v1.1.4

タイムライン上での音量キーフレーム編集を追加したリリースです。

デモ: https://shunsukesaito00.github.io/mvmake/

## 追加

- **音量キーフレームのタイムライン UI (MVP)**: AudioClip / VideoClip のクリップ上に音量カーブとハンドルを表示。ドラッグで時間・音量を編集、ダブルクリックでキーフレーム追加

## 改善

- **Slider のアクセシビリティ**: レンジ入力に `aria-label` を付与し、E2E・スクリーンリーダー対応を強化
- **E2E**: タイムライン上の音量キーフレームドラッグ編集シナリオを追加（全 18 件）

## 運用上の注意

- main への **連続 push は避け**、1 回の push ごとに Deploy ワークフローの完了（build → deploy → smoke）を待ってから次を push してください
