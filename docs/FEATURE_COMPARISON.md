## テストカバレッジ

- ユニットテスト (Vitest): 90件。ストア・クリップ・音量キーフレーム・書き出し・タイムライン UI・カラールック・映像フェード・書き出し解像度・テキストレイアウト・ウェディングテンプレ・写真ガイド・テキスト背景・章マーカー書き出し・マーカー編集
- E2E (Playwright): 29シナリオ（`e2e/basic.spec.ts` 3 + `e2e/editor.spec.ts` 26）
  1. 基本フロー(起動 → オンボーディング → テキスト追加 → タイムライン確認)
  2. サンプルプロジェクトをオンボーディングから開く
  3. プロジェクト一覧モーダルの開閉と新規プロジェクト作成
  4. インスペクターでのテキスト編集がタイムラインへ反映
  5. インスペクター未選択時のクイックスタートからテキスト追加
  6. モーダルのフォーカストラップと Escape で閉じる
  7. 「?」キーでショートカット一覧をトグル表示
  8. 画像インポート → クリップ配置 → トランジション適用
  9. クリップのドラッグ移動
  10. 右端ハンドルのトリム(ラベル重なり領域での回帰ケース含む)
  11. 左端ハンドルのトリム(開始位置と長さの同時変更)
  12. Alt+ドラッグでのクリップ複製と undo
  13. 書き出しスモーク(対応環境は MP4 ダウンロード、非対応環境はエラー表示)
  14. キーボードショートカット(Space 再生・停止、Cmd/Ctrl+Z)
  15. 音量キーフレームの追加・編集(オーディオクリップ)
  16. 本番スモーク用音量キーフレーム(`basic.spec.ts`、deploy smoke / `test:e2e:prod` と共通)
  17. 書き出しプリセットの保存と適用
  18. タイムライン上の音量キーフレームドラッグ編集
  19. カラールックプリセットの適用
  20. 映像クリップのフェードイン設定
  21. 正方形プロジェクトのネイティブ解像度書き出し UI
  22. インスペクターでの複数行テキスト入力
  23. 縦型9:16プロジェクトのネイティブ解像度書き出し UI
  24. 構造化ウェディングテンプレートの章マーカーと写真ガイド配置
  25. インスペクターでのテキスト行間・縦配置設定
  26. 写真ガイド区間へのスライドショー一括配置
  27. インスペクターでのテキスト字幕帯設定
  28. 書き出しダイアログでの章マーカー区間 In/Out 設定
  29. マーカーのインスペクター編集とタイムライン上ドラッグ移動

### 本番スモーク E2E (`npm run test:e2e:prod` / deploy smoke)

GitHub Pages 公開環境に対して `e2e/basic.spec.ts` の 3 シナリオを実行する。

1. 基本フロー(起動 → オンボーディング → テキスト追加 → タイムライン確認)
2. サンプルプロジェクトをオンボーディングから開く（再生ガイド含む）
3. 音量キーフレーム（オーディオインポート → インスペクターでキーフレーム追加）

## スコープ外

| 機能 | 理由 |
|------|------|
| AI字幕/自動編集 | サーバーレス方針のため |
| モーション追跡 | 実装コスト過大 |
| マルチカム | 用途外 |
| リアルタイム共同編集 | サーバーレス方針のため |
