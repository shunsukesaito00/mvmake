# 有名動画編集アプリ vs FABLE 機能比較

最終更新: 2026-07-11（v1.23.0 リリース）

## プロダクト目標

**結婚式ムービーが作りやすいブラウザ完結型エディタ**を目指す。CapCut / **Premiere Pro クラスのタイムライン編集・操作性**を婚礼制作フロー（構成→素材投入→テロップ/BGM→仕上げ→章ごと書き出し）に最適化する。

| 観点 | 目標 | v1.23.0 時点の評価 |
|------|------|-------------------|
| **婚礼ワークフロー** | テンプレ・章・写真ガイド・SRT・章 ZIP など一連が途切れない | **達成** |
| **Premiere コア編集** | マルチトラック・トリム/分割・キーフレーム・In/Out・マーカー | **おおむね同等** |
| **Premiere 操作性** | J/K/L・リップル編集・タイムライン上キーフレーム操作 | **おおむね同等** |
| **Premiere 仕上げ** | Lumetri 色調・多数トランジション・速度ランプ・調整レイヤー | **改善中** — ルック10種・トランジション11種・速度KF・調整レイヤー |
| **総合（婚礼用途中心）** | Premiere で婚礼ムービーを作る体験に近づける | **約 76〜80%** |

> 「同等」とは婚礼ムービー制作で**日常的に使う機能・操作**が Premiere と同程度使えること。AI 自動編集・マルチカム・共同編集などスコープ外は対象外（[スコープ外](#スコープ外)参照）。

## スコープ外

| 機能 | 理由 |
|------|------|
| AI字幕/自動編集 | サーバーレス方針のため |
| モーション追跡 | 実装コスト過大、婚礼用途では低頻度 |
| マルチカム | 用途外 |
| リアルタイム共同編集 | サーバーレス方針のため |
| ネストシーケンス | 調整レイヤーで簡易代替を検討。フル実装はコスト大 |
| プロキシワークフロー | ブラウザ完結・IndexedDB 方針と相性が悪い |
